# bir2307_meets_python

Automates the generation of BIR Form 2307 by combining Python scripting with PDF automation tools.

## Features
- Auto-fills BIR 2307 form from structured data
- GUI interface for ease of use
- Outputs fillable PDF ready for submission

## Built With
- [Python](https://www.python.org/) – core scripting logic
- [Tkinter](https://docs.python.org/3/library/tkinter.html) – GUI for user input
- [pdffiller](https://pypi.org/project/pdffiller/) – PDF form filling automation
- [Sejda](https://www.sejda.com/) – creates fillable PDF templates online
